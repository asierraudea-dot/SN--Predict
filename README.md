
# SN Predict 🚀

**Aplicación predictiva inteligente** para la planificación de programas de formación.

Predice el número de aprendices esperados según municipio, programa, sector y otras variables históricas.

---

## ✨ Características

- Predicción en tiempo real del número de aprendices
- Dashboard interactivo con tendencias
- Análisis histórico completo
- Recomendaciones por municipio
- Interfaz moderna y 100% en español

---

## 🛠️ Cómo Ejecutar

```bash
# 1. Clonar repositorio
git clone https://github.com/tuusuario/sn-predict-app.git
cd sn-predict-app

# 2. Instalar dependencias
pip install -r requirements.txt

# 3. Ejecutar
streamlit run app.py
